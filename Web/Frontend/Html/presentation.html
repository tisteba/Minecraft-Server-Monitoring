À propos du projet

Minecraft Server Monitoring est une application web permettant de surveiller et de gérer facilement votre serveur Minecraft depuis n'importe quel navigateur. Notre solution offre une interface moderne et intuitive qui vous donne un contrôle total sur votre serveur, tout en visualisant son état et ses performances en temps réel.

Ce projet a été développé dans le cadre de notre formation à YNOV, avec pour objectif de créer un outil professionnel qui facilite la gestion de serveurs Minecraft pour les administrateurs de tous niveaux.

Voir le projet sur GitHub

Fonctionnalités principales

Notre application offre une large gamme de fonctionnalités pour répondre à tous vos besoins de gestion de serveur :

Console en temps réel

Accédez à la console de votre serveur et envoyez des commandes directement depuis l'interface web.

Monitoring des ressources

Visualisez l'utilisation du CPU et de la RAM avec des graphiques dynamiques en temps réel.

Contrôle du serveur

Démarrez, arrêtez ou redémarrez votre serveur en un seul clic depuis n'importe où.

Gestion des fichiers

Téléchargez facilement des plugins, des mods ou des mondes sur votre serveur.

Sécurité avancée

Authentification sécurisée et protection contre les accès non autorisés.

Design responsive

Interface adaptée à tous les appareils pour une gestion en déplacement.

Technologies utilisées

Notre application repose sur un stack technologique moderne et robuste :

Frontend

HTML5
CSS3
JavaScript
Chart.js

Backend

Node.js
Express
Socket.IO

Script

Bash
Shell
Firewall

L'application utilise une architecture client-serveur avec une API RESTful pour les opérations standards et des WebSockets pour les communications en temps réel comme la console et les métriques de ressources.
